# 🤟 Indian Sign Language Translator  
### AI-Powered Bidirectional Translation & Learning System

An **AI-powered web application** designed to bridge the communication gap between **deaf/mute individuals and the hearing community** using **Indian Sign Language (ISL)**. The system enables **real-time sign-to-text/voice translation** and **text/voice-to-sign conversion**, along with an interactive **9-level ISL learning module**.

---

## 📌 Project Overview

The **Indian Sign Language (ISL) Translator** is developed using **Artificial Intelligence, Computer Vision, and Deep Learning** to provide real-time communication support.

The application supports:

✅ **Sign → Text/Voice Translation**  
✅ **Text/Voice → Sign Translation**  
✅ **Real-Time Gesture Recognition**  
✅ **9-Level Gamified Learning Module**  
✅ **User Authentication & Progress Tracking**  
✅ **PDF Certificate Generation**

## 🧠 AI Model

The system uses **MobileNetV2 CNN** for gesture classification.

### Why MobileNetV2?
- Lightweight architecture
- Fast real-time inference
- Low memory usage
- Suitable for CPU-based systems

### Model Details
- **36 Classes**
  - A–Z Alphabets
  - 0–9 Numbers
- **Accuracy:** ~88%
- **Input Size:** 224 × 224 × 3
